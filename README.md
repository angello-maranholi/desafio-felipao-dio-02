# Classificador de Nível de Herói

Projeto desenvolvido como parte de um desafio da DIO, com o objetivo de praticar lógica de programação em JavaScript.
O programa calcula o saldo de vitórias de um herói e determina seu nível com base nesse valor.
--------------------------------------------------------------------------------------------------------------------------------
Objetivo:
Aplicar conceitos básicos de JavaScript, como:
- Variáveis
- Funções
- Condicionais
- Operadores matemáticos

Tecnologias utilizadas:
- JavaScript
- Node.js
- Git e GitHub

Lógica do projeto:
- O número de vitórias e derrotas é definido pelo usuário
- O saldo de vitórias é calculado pela diferença entre vitórias e derrotas
- O herói recebe um nível conforme o saldo obtido

Classificação de níveis:
- Até 10 vitórias: Ferro  
- De 11 a 20: Bronze  
- De 21 a 50: Prata  
- De 51 a 80: Ouro  
- De 81 a 90: Diamante  
- De 91 a 100: Lendário  
- A partir de 101: Imortal  

Exemplo de saída: 
- O Herói tem saldo de 19 e está no nível Prata.
